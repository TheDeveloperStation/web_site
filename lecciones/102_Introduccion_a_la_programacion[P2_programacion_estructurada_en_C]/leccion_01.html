1- HISTORIA DE C Y LA PROGRAMACION ESTRUCTURADA


1.1 IMPORTANCIA DE C

El lenguaje C es uno de los lenguajes de programación más básicos que puedes encontrar a la hora de querer programar, lo cierto es, que si estas interesado en el mundo de la programación, una de las mejores opciones para empezar a practicar es C, esto debido al hecho de que entre sus características está el hecho de que comparte su sintaxis con posteriores lenguajes que fueron saliendo, como puede ser C++, C#, JAVA, PHP o JAVASCRIPT, además aprendiendo C podremos comunicarnos con sistemas operativos como Linux, Windows, Android, entre otros. Así mismo, en proyectos de robótica que puedan estar construidos con Arduino o con algún modelo como puede ser el Raspberry Zero, aprender C es muy útil al momento de elaborar el código que utilizara el robot en cuestión, e incluso este lenguaje puede ser enlazado con una base de datos SQL permitiéndonos tener un sistema completo como puede ser un punto de venta para alguna tienda ya que este lenguaje se comunica con el hardware.

1.2 QUE ES LA PROGRAMACION ESTRUCTURADA

Cuando hablamos de programación estructurada hablamos de un conjunto de técnicas para el desarrollo de algoritmos los cuales sean fáciles de escribir, leer, verificar y modificar, para lo cual nos ayudaremos de un diseño descendente y estructuras básicas. La programación estructurada fue propuesta por Böhm y Jacopini con la idea de crear programas más sencillos y rápidos, con el fin de disminuir la complejidad de las pruebas y testigo para ponerlos en funcionamiento. Como curiosidad, el primer lenguaje de programación estructurada fue Algol 60.

1.3 HISTORIA DE C

La historia del lenguaje C empieza en 1970 , en los laboratorios de la empresa Bell en donde Dennis Ritchie & Brian Kernighan empiezan la creación de C, Ritchie era un colaborador de Ken Thompson quien en ese entonces trabajaba con el sistema operativo Unix y quien había sido quien creara el lenguaje B (que usaba el SO Unix), el objetivo de la creación de C era el tener un compilador que pudiera garantizar la velocidad y consistencia en cada terminal asi como entender mejor el sistema operativo Unix, es decir, tener un SO más eficiente y que no sufra ralentizaciones. Es gracias a esto, que se pudo reescribir el sistema operativo Unix gracias a C (el cual además es la base de otros sistemas operativos que usamos aun en la actualidad), así para 1973, el 90% del SO Unix se había reescrito en lenguaje C, y para 1974 los laboratorios Bell conceden la licencia a distintas universidades iniciando así la distribución de C, el cual posteriormente en 1983 gracias a ANSI (una de las organizaciones de normalización más importantes de Estados Unidos), formaría un comité con el fin de convertir a C en un estándar, lo cual lo llevaría a ser un lenguaje de programación que se mantiene bien conservado hasta hoy en día gracias a su eficiencia, seguridad y optimización. Entre las características de C están: -Es un lenguaje estructurado. -Es de nivel medio, lo que quiere decir que es menos abstracto que otros lenguajes, pero posee ciertas características de un lenguaje de nivel alto. -Tiene buen rendimiento a la hora de cuidar los recursos que utiliza. -Versátil: en múltiples plataformas se puede programar en C.

1.4 POR QUE ESTUDIAR C

Una de las razones que hay para empezar por C es el hecho de que es uno de los lenguajes más populares y usados en el mundo, lo que lo lleva a poseer una gran variedad de librerías que se pueden usar para facilitar la elaboración de nuevos programas, su sencillez permite crear programas que sean de alto rendimiento en computadores de poca memoria. También al ser un lenguaje que tiene una programación estructurada lo vuelve visualmente más entendible, perfecto para empezar a practicar y desarrollar lógica al momento de crear programas, lo cual si se es un novato resulta bastante atractivo.

1.5 ELEMENTOS DE UN CODIGO (REPASO) VARIABLES, CONSTANTES, PALABRAS RESERVADAS

Ahora veremos algunos de los elementos de un código para repasar lo visto en la sección pasada de introducción a la programación: 1- Palabras reservadas: 2- Identificadores: Nombre que tendrán las variables y constantes en el programa (se recomienda que hagan referencia al dato que guardamos en ellas) 3- Caracteres especiales: son los signos “; () [] {}” que podemos encontrar para distintos usos, por ejemplo, el “;” para indicar que termino una instrucción, los {} para indicar dónde empieza y termina una estructura de selección, cíclica o un método o los “()” en donde especificaremos los parámetros que se necesiten para el funcionamiento del método. 4- Constantes: datos los cuales tienen un valor que no cambia. 5- Variables: datos los cuales pueden valer un distinto valor según lo requiera el programa. 6- Expresiones: combinación de variables y/o constantes para ejecutar una operación. 7- Instrucciones: sentencia o pasos a seguir de un programa.
