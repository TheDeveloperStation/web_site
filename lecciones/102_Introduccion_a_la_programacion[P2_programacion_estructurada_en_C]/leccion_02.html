2- SOFTWARE PARA PROGRAMAR

2.1- Que es un programa.

Un programa informático es una serie de instrucciones y procesos creados con el fin de cumplir una tarea específica en una computadora o sistemas de computadoras, estos pueden ser programas preinstalados en el sistema operativo o instalados por el usuario, los programas necesitan usar recursos de la computadora (CPU, RAM, Almacenamiento en el disco duro), así mismo, varios programas tienen la capacidad de modificarse mediante actualizaciones las cuales pueden agregar nuevas funciones, arreglar errores u optimizar su uso entre otros ejemplos. Estos pueden ser catalogados en: Software del sistema, los cuales son todos aquellos programas que utiliza el sistema operativo para funcionar, como un explorador de archivos, un programa que se encarga de comunica a los diferentes periféricos (mouse, teclado, micrófono, etc) con la computadora, líneas o consola de comandos, paneles de control del sistema operativo, etc. Software de aplicación, los cuales son aquellos programas secundarios que podemos descargar o instalar en una computadora y que permiten realizar distintas funciones, como puede ser desde una calculadora o editor de texto, pasando por un navegador, reproductores de música, videojuegos entre otros.

2.2- Que es un compilador.

El compilador es aquel software que nos ayuda a convertir un código de programación (la parte entendible por el programador) a un lenguaje máquina, este generalmente toma el código que hemos escrito, lo convierte a lenguaje ensamblador y luego lo traduce a lenguaje máquina. El termino compilador fue dado a principios de 1950 por Grace Brewster Murray Hopper, una científica informática y contraalmirante de la Marina de los Estados Unidos. Siendo esta pionera en la creación de herramientas relacionadas con el compilador, así mismo ayudo a popularizar la idea de los lenguajes de programación independientes de la maquina con lo cual empieza el desarrollo de COBOL, el cual es un lenguaje de programación de alto nivel que aún se utiliza a día de hoy.

2.3 –Que es un IDE y un editor de texto.

Un IDE es un entorno de desarrollo integrado, el cual es un programa que facilita al programador con el desarrollo de código de una manera eficiente, combinando las características de editar, crear, compilar y empaquetar software en un solo programa, entre alguna de las ventajas de un IDE está el hecho de que tiene automatización de código, resaltado de sintaxis, automatización de la creación local, compilación, depuración, entre otras. Algunos de los IDE que podemos encontrar actualmente están por ejemplo Visual Studio Code, Eclipse, Netbeans, Apple Xcode, Atom, Android Studio entre otros. Por otra parte, no siempre es necesario un IDE para la elaboración de un código, pues además de estos, también existen los editores de texto, los cuales nos ayudan a escribir texto plano el cual puede ser guardado con alguna extensión de programación como, por ejemplo .c, .cpp, .java, etc. Entre los editores de textos podemos encontrar uno que ya está por defecto en todas las computadoras, el blog de notas, aunque también hay otras opciones que podemos instalar como puede ser Sublime Text, CoffeeCup, Vim, o uno de los más populares, Notepad++, a diferencia de los IDE, los editores de textos poseen menos características como por ejemplo no suelen incluir un compilador, por lo que necesitaremos tener algún compilador externo al editor que usemos, sin embargo, son útiles al momento de elaborar distintos tipos de lenguaje, además, no utilizan tantos recursos como un IDE por lo que en equipos de bajos recursos suelen ir más rápido.

2.4- Programación en MS-DOS (Turbo C).

Turbo C es un IDE de programación para lenguaje C (y C++ posteriormente con Turbo C++), desarrollado para MS-DOS, el cual está basado en el IDE de Turbo Pascal, Turbo C fue conocido por su tamaño compacto, rápida velocidad de compilación y bajo precio en su momento, este, incluía un editor de código fuente, compilador de archivos y un vinculador, la primera versión fue lanzada el 13 de mayo de 1987.

2.5- Programación para este curso (Notepad++).

Notepad++ es un editor de texto gratuito de código libre, el cual sirve para la elaboración de texto plano, el cual puede guardarse en archivos con distintas extensiones para obtener así un código el cual podemos compilar, entre algunas de sus características podemos encontrar que tiene resaltado de sintaxis, auto completado, soporta distintos lenguajes de programación, permite hacer distintas codificaciones como ANSI, UTF-8, UCS-2, abrir múltiples documentos a la vez e instalación de plugins. Entre algunos de los lenguajes soportados por Notepad están: C, C++, COBOL, HTML, JS, CSS, PHP, JAVA, MATLAB, Objetive C, Pascal, Perl, Python, Ruby, SQL, XML, entre otros.
